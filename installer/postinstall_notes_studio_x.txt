 First Launch
---------------------------------------------------------------------
1. Launch SupermixStudioX from the Start menu or Desktop icon.
2. The application opens in your default web browser.
3. Open the "Bench" tab to review the bundled V48 benchmark results.

---------------------------------------------------------------------
 Configuration
---------------------------------------------------------------------
Models are stored in:
  %LOCALAPPDATA%\supermix_studio\models\

To add or update models, copy supported model artifacts there and restart.
